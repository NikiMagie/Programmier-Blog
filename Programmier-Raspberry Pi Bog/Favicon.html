So fügst du ein Favicon auf deiner Website ein

Was ist ein Favicon?


Ein Favicon ist das Logo oder das Symbol, dass links neben dem Tabnamen ist/steht.


So fügst du ein Favicon auf deiner Website ein

Als erstes musst du dir ein Logo erstellen und dieses anschließend in einer Quadratischendatei speichern.
Es ist empfelenswert, wenn du eine PNG-Datei von deinem Logo hast da diese von fast allen Browsern akzeptiert werden.

Danach gehst du am Besten auf die Website realfavicongenerator.net/. Dort kannst du dann deine Favicondatei hochladen. Wenn du deine Datei dort hochgeladen hast, kommst du zu einem Menu, auf dem du sehr viele Einstellungsmöglichkeiten hast.
Als erstes kannst du das Design für den Desktop Browser individualisieren.
Anschließend für das IOS Home Menu und für das Android Menu.
Als letztes kannst du noch optional ein paar Anpassungen bei dem Windows Metro Menu machen.
Ein wiederum entwas wichtigeres Feld ist die Bearbeitung von dem macOS Safari Browser. Du hast dort mehr möglichkeiten deine Website zu individualisieren.

Wenn du nun all diese Einstellungen nach deinen Wünschen bearbeitet hast, kannst du ganz unten auf den button: "Generate your HTML code" klicken.
Du erhälst nun bei Punkt eins eine Datei, die du in deinem Code einfügen musst. Nicht wundern, in deinem herruntergeladenem Ordner sind viele Dateien. Der letzte Schritt ist jetzt der, dass du den Angezeigten Code in Punkt drei in deinen head Bereich einfügen musst. Denk hierbei aber daran, dass du den Code nicht nur in deiner Index.html Datei einfügst, sondern auch in all einen Anderen HTML-Dateien. Sonst wird der Code nur auf deiner Startseite angezeigt und beispielsweise nicht auf deiner Impressum Seite.

Du kannst optional auch von anderen Seiten in Punkt vier schauen, wie diese Websiten in den unterschiedlichen Formaten aussehen.

Nun ist die Änderung fertig und du hast ein Favicon auf deiner Website.
